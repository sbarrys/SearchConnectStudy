> Search, Connect, Study 저희 프로젝트는 웹사이트상에서 스터디를 개설 및 참가를 원하는 모든 학우들에게 스터디 모임환경을 제공하기위해 만들어졌습니다.

## Install

```sh
cd frontend
npm run build
cd backend
npm install
```

## Run tests

```sh
cd backend
npm start
```
